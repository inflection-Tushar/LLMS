
1. Ollama: 
 Ollama is a software tool that allows you to run large language models (LLMs) locally on your computer. This means you can use powerful language processing capabilities without relying on cloud services or remote servers.

 System Requirement:
 - Operating system: Windows, Linux or Mac
 - Minimum 8GB Ram is Required to run 2b models
 - for 7b minimum 16GB Ram is Required

How to use?
- Download and install Ollama setup
- run or pull the required model using Ollama with the particular command
